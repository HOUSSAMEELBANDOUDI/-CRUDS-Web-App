# CRUD Web App

A simple CRUD (Create, Read, Update, Delete) web application built with vanilla **HTML**, **CSS**, and **JavaScript**. Demonstrates front-end data management without any frameworks.

## Tech Stack

- **HTML5**
- **CSS3**
- **JavaScript** (Vanilla)

## Features

- Create new records
- Read and display data
- Update existing records
- Delete records
- Clean and responsive UI

## How to Run

Open `index.html` in your browser.
